# AI Automation Roadmap

Este repositorio documenta mi ruta de aprendizaje desde cero en IA aplicada, automatización y herramientas técnicas.

## Objetivo

Convertirme en un perfil capaz de construir automatizaciones y sistemas de IA para empresas usando Claude, Make, n8n, APIs, Python, bases de datos y herramientas de contenido.

## Fases

* Fase 0: Preparación inicial
* Fase 1: Alfabetización tecnológica
* Fase 2: IA práctica y Claude
* Fase 3: Make
* Fase 4: n8n
* Fase 5: Python, APIs y bases técnicas
* Fase 6: RAG, agentes y sistemas de IA
* Fase 7: Cloud, seguridad y despliegue
* Fase 8: Vídeo, diseño y automatización de contenido

## Proyectos futuros

* Sistema de leads con IA
* Clasificador de emails
* Chatbot con documentos
* Automatización de reuniones
* Fábrica de contenido audiovisual
* Dashboard de operaciones

## Herramientas

Claude, Make, n8n, Python, GitHub, Google Sheets, Notion, Postman, APIs.
