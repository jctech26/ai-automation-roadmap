# Proyecto 00 — Entorno Profesional

## Objetivo

Preparar mi entorno profesional para estudiar IA aplicada, automatización, Claude, Make, n8n, APIs y Python.

Este proyecto documenta la preparación inicial de mi base de carrera: cuentas, herramientas, seguridad, carpetas, recursos, GitHub, LinkedIn, sistema de estudio y evidencias.

## Contexto

Este proyecto pertenece a la Fase 0 de mi ruta profesional hacia IA aplicada y automatización.

La Fase 0 no busca aprender herramientas avanzadas todavía, sino construir una base ordenada, segura y documentada para poder avanzar después hacia proyectos reales.
