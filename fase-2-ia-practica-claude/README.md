# Fase 2: IA práctica y Claude
 
## Objetivo
 
Aprender a utilizar Claude y otros modelos de lenguaje de forma práctica para resolver problemas, crear asistentes, analizar información y mejorar procesos de negocio.
 
## Temas que voy a estudiar
 
- Qué es un modelo de lenguaje
- Cómo escribir buenos prompts
- Cómo dar contexto a una IA
- Cómo pedir análisis, resúmenes y estructuras
- Cómo usar IA para procesos empresariales
- Cómo crear asistentes especializados
- Casos de uso de IA en ventas, soporte, marketing y operaciones
- Limitaciones y riesgos de la IA
 
## Herramientas
 
- Claude
